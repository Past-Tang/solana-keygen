## 概述

Solana 网络密钥对批量生成工具，基于 Ed25519 算法生成符合 Solana 标准的公私钥对，支持批量创建和导出。

## 功能特性

- **密钥对生成** -- 基于 Ed25519 算法生成 Solana 标准密钥对
- **批量创建** -- 支持一次性生成多个钱包
- **安全存储** -- 本地生成，私钥不外传
